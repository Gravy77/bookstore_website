This is an E-commerce website for a fictional space themed bookstore called 'Constellation Bookstore'.  
It was originally created for a class project in July 2019.

To get a sense of the website, please download and view the screenshots in screenshots.zip folder.

The web pages are:
Homepage
Category page
Cart page
Checkout page
Confirmation page

Details:
- Composed of Servlets and JSPs
- Language: Java
- Implements HTML/CSS/EL/JSTL/JavaScript for the user interface
- SSL enabled security
- Connected to MySQL (local) database using JDBC and Direct Access Objects
- Web accessible via Tomcat9

The original source code is in the src.zip folder.
